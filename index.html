¿Cómo usar esta plataforma?

1

Explora el menú

Abre cualquiera de las 35 tablas o 12 vistas desde el menú lateral. Puedes ocultarlo con el botón ☰ de arriba.

2

Busca variables

Usa el buscador de arriba a la derecha para encontrar cualquier variable y ver su definición oficial del INEGI.

3

Consulta el significado

Dentro de una tabla, pasa el cursor sobre el icono de información de cada columna para leer qué significa esa pregunta.

4

Filtra los datos

Con el botón Filtros acotas por valores (sexo, edad, entidad…). El conteo se actualiza al instante.

5

Exporta a CSV

Descarga los datos —con o sin filtros— con el botón Exportar CSV. La descarga respeta lo que tengas filtrado.

6

Vistas analíticas

Las vistas maestras ya cruzan varias tablas (salud + nutrición) y están listas para análisis directo.
